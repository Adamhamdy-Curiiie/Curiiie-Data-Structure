# 📚 Data Structures & Algorithms in C++
> A learning repository covering **Linked Lists** and **Sorting Algorithms**

---

## 🗂️ Repository Structure

```
📁 LinkedList/
│   ├── LinkedListBase.h           ← Abstract base class + node structs + iterators
│   ├── LinkedList.h / .cpp        ← Unordered singly linked list
│   ├── OrderedLinkedList.h / .cpp ← Sorted singly linked list
│   └── DoublyLinkedList.h / .cpp  ← Doubly linked list
│
📁 Sorting/
│   └── main.cpp                   ← All 8 sorting algorithms
```

## 🔀 Part 2 — Sorting Algorithms

### 🗺️ Learning Roadmap

```
① O(n²) — start here
   ├── Selection Sort
   ├── Bubble Sort
   └── Insertion Sort

② Bridge
   └── Shell Sort

③ O(n log n) — divide & conquer
   ├── Merge Sort
   └── Quick Sort

④ Non-comparison
   ├── Counting Sort
   ├── Radix Sort
   └── Bucket Sort
```
